## 📖 About the Project

The **Student Course Registration System** is a **Java-based console application** developed as part of the **CodSoft Java Development Internship (Task 5)**.

This application allows students to register for courses, manage their enrolled subjects, search available courses, view seat availability, and drop registered courses through an easy-to-use menu-driven interface.

The project demonstrates the practical implementation of **Object-Oriented Programming (OOP)** concepts and provides a structured solution for handling course registration in an educational environment.

Designed with multiple Java classes, the application is modular, reusable, and easy to maintain.

# 📚 Classes Used

## 📌 Main.java

- Starts the application
- Displays the menu
- Handles user interaction
- Controls program flow

---

## 📌 Student.java

Responsible for managing student information.

Functions include:

- Store Student ID
- Store Student Name
- Register Courses
- Drop Courses
- Display Registered Courses
- Student Summary

---

## 📌 Course.java

Represents every available course.

Contains:

- Course Code
- Course Name
- Description
- Capacity
- Available Seats

Functions:

- Register Student
- Drop Student
- Display Course Details

---

## 📌 CourseManager.java

Responsible for course management.

Functions:

- Add Courses
- Remove Courses
- Search Courses
- Display Available Courses
- Count Total Courses

---

## 📌 RegistrationSystem.java

Acts as the core controller.

Responsibilities:

- Load Courses
- Register Students
- Drop Courses
- Check Seat Availability
- Display Statistics
- Manage Student Operations
